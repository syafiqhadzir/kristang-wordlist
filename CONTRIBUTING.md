# Contributing to Kristang Wordlist

First off, thanks for taking the time to contribute!

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your machine.
3.  **Create a branch** locally with a succinct but descriptive name.
4.  **Make changes** to the wordlist or scripts.
5.  **Run the sort script** to ensure the wordlist is properly ordered:
    ```bash
    npm run sort
    ```
6.  **Commit changes** to the branch.
7.  **Push changes** to your fork.
8.  **Open a Pull Request** in our repository.

## Coding Standards

- We use [Prettier](https://prettier.io/) for code formatting.
- Ensure all CI checks pass before merging.

## Wordlist Guidelines

- Ensure words are spelled correctly according to Kristang orthography.
- if you are adding new words, please verify them against reliable sources.
